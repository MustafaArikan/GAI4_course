# Unit 5: Generative Adversarial Networks

Train Generative Adversarial Networks (GANs) with PyTorch and learn how to *evaluate*
a generative model that has no reconstruction target and no labels. 

## Vanilla GAN on MNIST

* **`00_PrepData_MNIST.ipynb`** — Download MNIST, visualize
* **`01_GAN_MNIST.ipynb`** — A fully-connected generator. Training loop, loss curves, sample snapshots.
* **`02_Evaluate_GAN_MNIST.ipynb`** — Evaluate the generator

## Usage

1. Run `uv sync` to install dependencies.
2. Run notebooks sequentially (`00 → 01 → 02`)
